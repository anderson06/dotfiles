# dotfiles

My personal dotfiles and tools setup

## Getting Started

These instructions will install the dotfiles on your machine

### Prerequisites

- Ubuntu 16.04
- git

### Installing

```
git clone git@github.com:anderson06/dotfiles.git
cd dotfiles
sudo bash install.sh
```
